\subsection{Statistiques descriptives}

\begin{frame}{2.1.1 Annotations du corpus CyberAggAdo utilisées}
\footnotesize
\begin{description}
  \item[EMOTION] émotion repérée dans l'énoncé : colère, dégoût, joie, tristesse, etc.
  \item[MODE] mode d'expression de l'émotion : suggérée, montrée, comportementale ou désignée.
  \item[ROLE] de l'autre : victim, bully, bully\_support, victim\_support ou conciliator.
  \item[TARGET] rôle de la cible du message.
  \item[HATE] degré d'agressivité du message : ouverte, couverte ou absente.
  \item[VAB] type de violence verbale : blâme, dénigrement, insulte, menace ou autre.
  \item[INT] intention du message : attaquer, défendre, apaiser, encourager, etc.
  \item[CTX] contexte interactionnel dans lequel l'énoncé répond ou s'inscrit.
\end{description}
\end{frame}

\begin{frame}{2.1.2 Explication de la signification des étiquettes 1}
\footnotesize
\begin{columns}[T,onlytextwidth]
\begin{column}{0.52\textwidth}
{\centering\textbf{ROLE / TARGET}\par}
\begin{description}
  \item[bully] personne qui initie le harcèlement.
  \item[victim] personne harcelée.
  \item[bully\_support] personnes qui aident le harceleur.
  \item[victim\_support] personnes qui aident la victime.
  \item[conciliator] personnes qui cherchent à médiatiser le désaccord.
\end{description}
\end{column}
\begin{column}{0.44\textwidth}
{\centering\textbf{HATE}\par}
\begin{description}
  \item[OAG] agressivité ouverte : hostilité explicite, menace, insulte ou haine directe.
  \item[CAG] agressivité couverte : agressivité masquée, indirecte ou implicite.
  \item[NAG] non agressif : absence de menace, insulte ou intimidation.
\end{description}
\end{column}
\end{columns}
\end{frame}

\begin{frame}{2.1.3 Explication de la signification des étiquettes 2}
\scriptsize
\begin{columns}[T,onlytextwidth]
\begin{column}{0.36\textwidth}
{\centering\textbf{VAB}\par}
\begin{description}
  \item[BLM] blâme : rendre la victime responsable de l'abus.
  \item[DNG] dénigrement : remarques blessantes et non constructives.
  \item[NCG] insultes ou name calling.
  \item[THR] menace visant à effrayer ou contraindre.
  \item[OTH] autre.
\end{description}
\end{column}
\begin{column}{0.60\textwidth}
{\centering\textbf{INT / CTX}\par}
\begin{description}
  \item[ATK] attaque : agression délibérée, moquerie, exclusion ou discrédit.
  \item[DFN] défense : réponse protectrice à une attaque.
  \item[CNS] contre-discours non agressif contre un discours nuisible.
  \item[AIN] incitation ou validation d'un message négatif.
  \item[GSL] gaslighting : minimisation ou invalidation du vécu de la victime.
  \item[EMP] empathie, compassion ou soutien.
  \item[CR] résolution de conflit, médiation ou désescalade.
  \item[OTH] intention peu claire, énoncé neutre ou fragmentaire.
\end{description}
\end{column}
\end{columns}
\end{frame}

\begin{frame}{2.2.1 Lecture de la notation}
\small
\begin{itemize}
  \item \(\rho(X=a,Y=b)\) reprend le coefficient Phi pour une paire de variables binaires.
  \item \(\chi^2(F, X=a)\) résume le test global entre une famille catégorielle \(F\) et une variable binaire.
  \item Les p-values très petites sont ramenées à des seuils : \(p<0.05\), \(p<0.01\), \(p<0.005\) ou \(p<0.001\).
  \item \(n_{\cap}\) indique le nombre de cooccurrences de la paire.
\end{itemize}
\end{frame}

\begin{frame}{3.2.1 Métriques utilisées}
  \vspace{1em}

  \vspace{0.5em}
  La précision mesure la fiabilité des prédictions positives :\par
  \vspace{0.5em}
  \colorbox{softSurface}{%
    \begin{minipage}{0.4\textwidth}
      \vspace{-0.2em}
      \[
      \text{Precision} = \frac{TP}{TP + FP}
      \]
      \vspace{-0.6em}
    \end{minipage}%
  }

  \vspace{2em}

  \vspace{0.5em}
  Le rappel mesure la capacité du modèle à retrouver les instances réellement positives :\par
  \vspace{0.5em}
  \colorbox{softSurface}{%
    \begin{minipage}{0.4\textwidth}
      \vspace{-0.2em}
      \[
      \text{Recall} = \frac{TP}{TP + FN}
      \]
      \vspace{-0.6em}
    \end{minipage}%
  }

\end{frame}

\begin{frame}{3.2.2 Interpréter les baisses de performance}
\small
\begin{itemize}
  \item Une baisse de précision suggère que certains indices lexicaux ou contextuels valides dans TTK deviennent trompeurs dans CyberAggAdoLarge.
  \item Une baisse de rappel suggère que l'émotion concernée est exprimée dans CyberAggAdoLarge par des formes lexicales, discursives ou contextuelles différentes de celles apprises sur TTK. EMOTYC n'ayant jamais vu ces formes, il ne les détecte pas ou les associe à tort à un autre label émotionnel.
\end{itemize}
\end{frame}

\begin{frame}{3.3.5 Points de vigilance}
  \begin{itemize}
    \item Les labels tr\`es rares dans CyberAggAdoLarge doivent \^etre interpr\'et\'es avec prudence : \textit{Admiration} a 0 occurrence positive, \textit{Embarras} 2, \textit{Fiert\'e} 3, \textit{Peur} 4.
    \item La couleur ne distingue pas le signe : une case claire peut aussi correspondre \`a une absence de cas suffisants pour comparer.
    \item La heatmap sert surtout \`a localiser les zones de non-transf\'erabilit\'e ; l'analyse qualitative des erreurs reste n\'ecessaire pour expliquer les m\'ecanismes.
  \end{itemize}

  \vspace{1em}
  \small
  Conclusion : EMOTYC apprend bien le domaine TextToKids, mais les formes discursives de CyberAggAdoLarge changent assez pour d\'egrader la g\'en\'eralisation, notamment sur les modes d'expression et certains labels \'emotionnels.
\end{frame}
